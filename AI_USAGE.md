# AI Usage Declaration

- Claude — used for initial code generation and ideas
- Visual Studio Code Debugger — used for identifying and resolving code errors

## How It Was Used
Claude was used to generate initial drafts of code across the src/ modules, which were 
then reviewed, tested, and modified by the group to ensure correctness and alignment 
with the assignment requirements. The VS Code debugger was used to identify and fix 
errors during development. All AI-generated code was critically evaluated and edited 
by group members before being incorporated into the final submission.

## Written Ourselves

was authored by the group independently. This includes all sleeve-level performance 
discussion, APRA check interpretation, attribution analysis, and key insights. The 
group made all analytical and methodological decisions independently.

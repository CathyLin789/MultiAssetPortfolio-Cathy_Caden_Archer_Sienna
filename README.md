## Repository Structure
- `data/` — Frozen teaching dataset (monthly manager and benchmark returns, risk-free rate, SAA weights)
- `src/` — Reusable Python modules (performance.py, apra_checks.py, attribution.py, data_loader.py)
- `notebooks/` — Main report notebook containing all analysis, charts, and discussion

## How to Run
1. Clone the repository
2. Install dependencies: `pip install pandas numpy matplotlib jinja2`
3. Open `notebooks/report.ipynb` in Jupyter or VS Code
4. Run all cells from top to bottom
